## Research Workflow

This skill guides the research process:

### Step 1: Define Research Question
- Clearly understand what needs to be researched
- Break down into specific questions if needed

### Step 2: Gather Information
- Use websearch for general information
- Use webfetch for specific URLs
- Use codesearch for code examples

### Step 3: Analyze and Synthesize
- Combine findings from multiple sources
- Identify patterns and commonalities
- Note conflicting information

### Step 4: Report Findings
- Summary of key findings
- Detailed analysis
- Recommendations or next steps

## Tools Used

- **websearch**: Search the web for information
- **webfetch**: Fetch specific URLs for detailed content
- **codesearch**: Find code examples and patterns

## Usage

When you need to research something:
1. Load this skill: `skill({ name: "research" })`
2. Define your research question
3. Let the workflow guide you through the process

## Tips

- Be specific about what you're researching
- Use multiple sources for verification
- Keep track of sources for citation
- Focus on actionable findings
